ingressCompagnion
=================
A project to simplify day 2 day gaming in Ingress

Short list of commands:

| Command | Description |
| --- | --- |
| `mvn spring-boot:run` | run the application (on localhost:8080)|
| `mvn test` | run all the tests |
| `mvn package` | export the project as a `WAR` |
